## **מסמך הטמעה moh-package 1.2.0**

## תקציר:

Package 1.2.0 תומך בנגישות.

עם זאת עדיין נדרשים שלבי הטמעה הכרחיים (לא רשות) לצורך התאמת הpackage שכולל את המנגנון הרוחבי בכל הcomponents, לאפליקציה המשתמשת בתשתית.

**באגים שטופלו:**

1. שימוש ב lazy loading

**היכולות שנוספו:**

1. נוסף רכיב לבחירת שפה
2. נוסף אירוע למשתמש ב cascading select , האירוע מתרחש בעת שנוי רשימת האפשרויות של אלמנט הבן.
3. נוסף אירוע לשרות של ה translate המתרחש כאשר מתחלפת שפה.
4. נוסף directive להזנת ספרות בלבד

## הטמעה:

1. אתחול של ה prefix ברכיב טלפון השתנה:
קודם היה צריך לשלוח אובייקט באופן הבא :  {Code: number, Text:string}
כעת יש לאתחל עם אובייקט מסוג Prefix, לדוגמה: new Prefix(1,&#39;050&#39;)
2. רכיב בחירת שפה מופיע ברכיב ה header באופן אוטומטי, כדי להסתיר אותו יש להוסיף attribute לרכיב moh-header : [showSelectLanguage]=&quot;false&quot;
3. כאשר משתמשים ברכיב ה select מסוג auto complete יש להגדיר באופן הבא **בדיוק** :

isAutoComplete=&quot;true&quot;

## תיקוני באגים:

1. תיקון שגיאה עבור שימוש ב lazy loading :
הסרת import של המודול BrowserAnimationsModule

## שימוש ביכולות:

**Select language**

הוספת רכיב לבחירת שפה,

הרכיב מציג את רשימת השפות מאומברקו, כאשר מחליפים שפה הרכיב אחראי על שינוי הכיוון של הדף בהתאם לשפה, ומעדכן את השפה הנוכחית בשרות התרגום

Selector: moh-select-language

Module: SelectLanguageModule

@Input:

1. labelTextKey:string –

מקבל את המפתח מאומברקו של הכותרת שתופיע מעל השדה

1.  currentLang: string –

מקבל את השפה הנוכחית (בשתי אותיות קטנות לדוג: אנגלית- en)
שתהיה מוצגת ברכיב כברירת מחדל,

אם לא נשלח ערך והוגדרה שפה לאתר ב app.component (חוץ מהשפה הדיפולטיבית), השפה שתוצג ברכיב כערך ברירת המחדל תהיה השפה המוגדרת ב app.component,

אם לא נשלח ערך ולא הוגדרה שפה לאתר לא יוצג ערך נבחר ברכיב.

כדי להגדיר את השפה בה ישתמש האתר

לאחר שורת הקוד (בה מגדירים את השפה הדיפולטיבית):

```typescript
this.translate.setDefaultLang(defaultLanguage);
```

יש להוסיף את השורה הבאה:
```typescript
this.translate.use(currentLanguage); //currentLanguage = 'he' / 'en'…
```

**Prefix class**

נוספה מחלקה המייצגת אובייקט של קידומת עבור רכיב טלפון,

המחלקה מכילה את המאפיינים הבאים:

code:number

text:string

וכן constructor המאתחל את המאפיינים של המחלקה:

```typescript
Constructor(code:number, text:string)
```

**Cascading select**

**Select component**

לרכיב  select  נוסף @Output  onChildDataChange
הארוע מתרחש במקרה שיש לרכיב parentSelect -

**נגישות**

רוב תיקוני הנגישות מוטמעים ברכיבי התשתית כך שכמעט לא נדרשים שינויים מצד הפרויקט המשתמש ב moh-package למעט הנקודות המפורטות למטה.

**Numbers Only Directive**

Directive זה הוא מסוג attribute. כאשר מוסיפים את אטריביוט זה לרכיב כלשהו, הוא מאפשר הזנת ספרות בלבד ברכיב.

יש להוסיף את DirectivesModule לרכיב המשתמש ב directive זה. את mohNumbersOnly  יש להוסיף כ attribute  לרכיב.

```typescript
import { DirectivesModule } from '../../directives/directives.module';
```
לדוגמא:

```html
<moh-textbox formControlName="Number" mohNumbersOnly></moh-textbox>
```

**Skip Nav Links Component**

Component זה מאפשר למשתמש לדלג לרכיבים עיקריים בדף בעזרת ניווט המקלדת.

```typescript
  @Input() skipToMainMenu: boolean = false; //האם לנווט לתפריט ראשי

  @Input() skipToSubMenu: boolean = false; //האם לנווט לתפריט משני

  @Input() skipToMainContent: boolean = true; //האם לנווט לתוכן ראשי
```
יש להוסיף את SkipNavLinksModule  ל - app.module
```typescript
import { SkipNavLinksModule } from './moh-package/components/skip-nav-links/skip-nav-links.module';
```
יש להוסיף ל app.component.html מעל תגית ה header את השימוש ברכיב.

לדוגמא:

```html
<moh-skip-nav-links [skipToMainMenu]="true" [skipToSubMenu]="true"></moh-skip-nav-links>
```

**Base Element**

נוסף:

```typescript
@Input() ariaLabelText: string
```
לכל אחד מהרכיבים של הטופס התווסף aria-label עם הטקסט של ה label.

במידה ואין label מעל הרכיב יש לשלוח ariaLabelText

לדוגמא:

```html
<moh-textbox formControlName="houseNumber" ariaLabelText="בית"></moh-textbox>
```


**Form Title Component**

 נוסף:

```typescript
  @Input() showRequiredFieldsMessage: boolean = true;
```

בכל טופס יוצג בכותרת טקסט שכל השדות המסומנים ב \* הם חובה.

ניתן להסתיר את הטקסט כך:

```html
<moh-form-title [showRequiredFieldsMessage]="false"></moh-form-title>
```
