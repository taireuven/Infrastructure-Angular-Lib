## **מסמך הטמעה moh-package 1.3.12**

## תקציר:

Package 1.3.12 מכיל תוספות ל-ag-grid, ועידכון FormArrayTemplateComponent

**באגים שטופלו:**

1. radio-button-table - שינוי מבנה ה-value של ה-form control.

**היכולות שנוספו:**

1. ag-grid - ביטול אופציה לנעיצת עמודה ע"י המשתמש, והעברת היכולת למפתח.
2. ag-grid - תיקוני עיצוב ל-desktop  ולמובייל.
3. ag-grid - כל הפרמטרים האופציונלים הועברו לאוביקט.
4. FormArrayTemplateComponent - נוסף Output בשם afterControlAdded שמופעל לאחר הוספת פריט למערך.
## הטמעה:	

## תיקוני באגים:

**RadioButtonTable**

נעשה שינוי במבנה ה-value של ה-FormControl, כעת הערך הוא המבנה של FormGroup:
```typescript
{
 quesitonId1:"answerValue1",
 quesitonId2:"answerValue2",
}

```

## שימוש ביכולות:

**Moh-ag-grid-service**

Methods

getMohLinkColumn() 

getMohColumn()

getMohSelectColumn()

getMohDateColumn()

מתודות אילו כבר קיימות, רק צורת השימוש בהן השתנתה.

לדוגמה:
```typescript
this.gridAgService.getMohSelectColumn('name', { arrayParams: ["David", "Sem", "Hello", "World"], editable: true, isPinned: false })

this.gridAgService.getMohLinkColumn('email', { translateCode: 'Email_Label', routerLink: "/wizard/address", arrayParams: ["name"] })

this.gridAgService.getMohDateColumn('brithDate', new colExtraDef({ translateCode: 'birthDate' })),

```

**FormArrayTemplateComponent**

1.	נוסף Output בשם afterControlAdded שמופעל לאחר הוספת פריט למערך.
דוגמא לשימוש:

Html:

```typescript
<moh-form-array-template [formArray]="demoForm.controls.formArray" [abstractControl]="absControl" [afterControlAdded]="itemAdded()">
    <ng-template #templateRef let-item>
     	<moh-textbox[formControl]="item.control.get('firstName')"></moh-textbox>
    </ng-template>
 </moh-form-array-template>
```
TS:
```typescript
itemAdded() {
    console.log('item added to array!');
}
```
