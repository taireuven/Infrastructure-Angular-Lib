## **מסמך הטמעה moh-package 1.3.15**

## תקציר:

גירסה זו מכילה תיקוני באגים, mini-footer, טיפול באייקונים, ותמיכה של רכיב ה header ברכיבי ההזדהות.

**באגים שטופלו:**

1. date-select - תיקון בדיקת תקינות התאריך
2. Translate - במקרה שהפנייה לאומברקו נכשלה, האפליקציה לא תיפול אלא תעלה ללא תרגומים.
3. check-box-group - הצגת כותרת לרכיב עפ"י הערך של ה  textKey / labelText המתקבלים ב Inputs.
4. header - פתיחת אפשרות לקישור חיצוני מהכפתור.

**היכולות שנוספו:**

1. date-select -  אפשרות להסתרת החודש ו/או השנה 
2. הוספת מודל Telephone המותאם לערך של הרכיב moh-telephone.
3. חשיפת השרות UtilsService והפונקציה Initialize המשמשים לאתחול ערכי הקונפיגורציה.
4. החלפת כל mat-icon בתגית i ומעבר לשימוש באייקונים דרך Icomoon (פירוט במסמך נפרד)
5. mini-footer - הסתרת האייקונים הגדולים בפוטר והשורה הקטנה מצד שמאל
6. חשיפת ה-URL של השירות FileUpload לדריסה ע"י פרויקט חיצוני.
7. header v2 - הוספת ng-content עבור קוביית לוגין.
8. header v2 - הוספת input institutionName עבור ההזדהות להצגת שם המוסד במובייל.

## שימוש ביכולות:

**DateSelect Component**

נוספו @Inputs :
```typescript
  /**
  * Whether to display month select or not
  */
  @Input() displayMonth: boolean = true;
  /**
  * Whether to display year select or not
  */
  @Input() displayYear: boolean = true;
```

אם לא נשלח ערך ל Inputs הנ"ל יוצגו השדות של החודש והשנה,  
כדי להסתיר את השדה המבוקש יש לשלוח את הערך false ל input המתאים.

**MOH icons**

השימוש באייקונים באתר נעשה תוך שימוש בתגית i ואפליקציית icomoon.

[הסבר לשימוש](http://mohdigitaltest.health.gov.il/additional-documentation/moh-icons.html)

**Footer component**

נוספו ארבעה inputs חדשים לצורך הצגת מיני פוטר + יצירת קשר המאפשר הכנסת טקטס, אייקון וקישור
```typescript
  /**
  * Whether to show mini footer
  */
  @Input() miniContactState: boolean = false;
  /**
  * icon name for contact us
  */
  @Input() miniContactIcon?: string = "contact_us";
  /**
  * text to show under icon
  */
  @Input() miniContactTextKey?: string = 'contactUsMenu';
  /**
  * link to contact us form/mail
  */
  @Input() miniContactLink?: string = "https://www.health.gov.il/UnitsOffice/Pages/UnitsLobby.aspx";
```

**Fileupload component**

רכיב ה-File upload פונה ל- Upload API של התשתית, ללא אפשרות לשנות אותו.

בגירסה זו נוסף מפתח חדש לקובץ ה-config.json שבו ניתן להכניס כתובת של כל API שמבצע העלאת קבצים ובכך לדרוס את הפניה ל-API של התשתית.
```typescript

  "fileUploadApiURL": null

```

**header v2 component**

נוסף לרכיב ng-content המאפשר לשלוח תוכן בין תגיות האלמנט moh-header.  
יכולת זו נוספה עבור תשתית ההזדהות להוספת רכיב קוביית הלוגין ל header.  
כדי לאפשר יכולת זו יש להוסיף בקובץ ה config תחת ה section של moh-package: 
```json
"enableCustomHeader": true
```
 דוגמה לשימוש:
 ```html
 <moh-header>content...</moh-header>
 ```
