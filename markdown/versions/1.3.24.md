## **מסמך הטמעה moh-package 1.3.24**

## תקציר:

גירסה זו מכילה תיקונים ויכולות חדשות.

**באגים שטופלו:**

1. **עיצוב משובש לאלמנט בפוקוס באקספלורר** - הסרת המסגרת במצב focus באקספלורר.
2. **wizard-route** 
    - תיקון באג מגירסה 1.3.23 - במקרה שלא משתמשים ביכולת החדשה של הצגת הודעה לפני מעבר לשלב הבא.
    - תיקון עבור אפשרות להשתמש בכמה רכיבי wizard באפליקציה אחת (יש לממש לפי הפירוט למטה).
3. **date-picker** - תיקון באג מגירסה 1.3.23 - במקרה שלא נשלח מערך של ולידציות.
4. **date-select** - הצגת ההודעה של שדה החובה בהתאמה לשדות המוצגות ברכיב.
5. **header** - תיקון עיצוב הכפתור - באג מגירסה 1.3.23.
6. **password** - החלפה בין האייקונים של העין.

**היכולות שנוספו:**

1. **select** - נוספה יכולת להציג חלק מהערכים לא מאופשרים.
2. **modal service** - נוספה תמיכה להצגת כפתור ביטול בהודעת הצלחה.
3. **wizard-route** - נוספה יכולת להציג\להסתיר את כפתור המעבר לשלב הקודם עבור כל שלב בנפרד. 
4. **radiobutton-group** - הצגת הטקסט של ה options כטקסט עשיר.

## שימוש ביכולות:

**select**

נוספה תמיכה ביכולת לא לאפשר ערכים מסוימים מהרשימה המוצגת ברכיב.  
יכולת זו נתמכת במטריאל וכעת היא מוחצנת בהתאמה גם ברכיב שלנו.

כדי לא לאפשר ערך מסוים יש לשלוח את הרשימה כמערך של אוביקטים ובאוביקט הרצוי להוסיף את המאפיין disabled, לדוג':
```typescript
  optionsList: Observable<any[]> = Observable.of([
    { Id: 1, text: 'aa' ,disabled:true},
    { Id: 2, text: 'bb' },
    { Id: 3, text: 'cc' },
  ]);
```

**modal service**

נוספה תמיכה להצגת כפתור ביטול בהודעת הצלחה.  
השינוי נעשה בפונקציות : successMsg, successMsgWithTranslation, 

לפונקציה successMsg נוספו הפרמטרים הבאים: 

```typescript
showCancelButton: boolean = false
cancelButtonText: string = "Cancel"
```
לפונקציה successMsgWithTranslation נוספו הפרמטרים הבאים: 
```typescript
showCancelButton: boolean = false
cancelButtonTextKey: string = "cancel"
```

חתימות הפונקציות:
```typescript
  public successMsg(message: string, title: string = "Success", confirmButtonText: string = "OK", showCancelButton: boolean = false, cancelButtonText: string = "Cancel", messageParams?: any, titleParams?: any, allowEscapeKey: boolean = true, allowOutsideClick: boolean = true, customClass?: string, reverseButtons?: boolean): Promise<any> 

  public successMsgWithTranslation(messageKey: string, titleKey: string = "success", confirmButtonTextKey: string = "ok", showCancelButton: boolean = false, cancelButtonTextKey: string = "cancel", messageParams?: any, titleParams?: any, allowEscapeKey: boolean = true, allowOutsideClick: boolean = true, customClass?: string, reverseButtons?: boolean): Promise<any>
```
דוגמה לשימוש:
```typescript
this.modal.successMsg("הודעה ההצלחה", "כותרת ההודעה", "אישור", true, "ביטול");
```

**wizard-route**

1. נוספה אפשרות להשתמש במופעים רבים של רכיב wizard ב module אחד.  
לשם כך נוסף ה input הבא:

    ```typescript
      @Input() wizardId?: string;
    ```
    כאשר משתמשים במופעים רבים של רכיב wizard חובה לשלוח ב input זה מזהה יחודי לכל מופע, לדוגמה:
    ```html
      <moh-wizard-route [steps]="steps" (onSubmit)="submitWizard($event)" wizardId="firstWizard"></moh-wizard-route>
    ```
    כמו כן הפונקציה WizardDataService.getFormData מקבלת פרמטר חדש:
    ```typescript
      getFormData(wizardId?: string);
    ```
    במקרה שיש צורך בשליפת הנתונים של ה wizard במקום כלשהו באפליקצייה, ניתן לעשות זאת באמצעות הפונקציה הזו באופן הבא:

    ```typescript
      this.wizardDataService.getFormData("firstWizard");
    ```
2. נוספה יכולת להציג\להסתיר את כפתור המעבר לשלב הקודם - עבור כל שלב בנפרד.  
    כדי להגדיר עבור כל הwizard האם להציג את הכפתור יש לשלוח ערך מתאים לinput allowPrev, ערך ברירת המחדל הוא true - מציג את הכפתור.

    במקרה שרוצים להסתיר את הכפתור הזה רק בחלק מהשלבים - אין צורך לשלוח ערך לinput הנ"ל, אלא יש לתת ערך false למאפיין allowPrev  - בתוך הרכיב המייצג את השלב הרצוי היורש מהמחלקה WizardStep:
    ```typescript
    export class stepWithoutPrevButton extends WizardStep implements OnInit {

      constructor(private fb: FormBuilder, injector: Injector) {
        super(injector);
        this.stepFormGroup = this.fb.group({
          firstName: ['', mohValidators.required()],
          lastName: [''],
        });

        this.allowPrev = false;
      }
    }
    ```
