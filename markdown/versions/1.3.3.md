## שימוש ביכולות:

**Header**

נוסף Input homePath@

מקבל מסלול של ה route אליו ינווט הכפתור &quot;חזרה לדף הבית&quot;.

ברירת החדל היא: &quot;/&quot; – הדף הראשי של האפליקציה.

**Textbox, Password**

ב textbox התווסף:

```typescript
@Output() focus: EventEmitter<any>
```

ב password התווספו:

```typescript
@Output() focus: EventEmitter<any>
@Output() blur: EventEmitter<any>
```

ה outputs מחצינים את האירועים focus ו- blur של הרכיבים.

**דוגמא לשימוש:**

**Html:**
```html
<mph-textbox formControlName="firstName" (focus)="onFocus()" (blur)="onBlur()"></moh-textbox>
<mph-password formControlName="password" (focus)="onFocus()" (blur)="onBlur()"></moh-password>
```
 
**TS:**

 ```typescript
onFocus(){
    console.log("focus");
}

onBlur(){
    console.log("blur");
}
 ```

**Modal Service**

הוחצנו שני פרמטרים אופציונליים נוספים בכל אחד מפונקציות ה modal:

- customClass –שם של class של css שיעטוף את ה .modal ע&quot;י שליחת הפרמטר הזה, ניתן לשלוט על העיצוב של ה modal הספציפי ע&quot;י הגדרת css בתוך שם ה- class שישלח ל- modal.
- reverseButtons – ערך בוליאני האם להציג את הכפתורים של הmodal  בסדר הפוך.

דוגמא לשימוש:
```typescript
this.modal.confirmMsg("msg", "title", "yes", "no", null, null, false, false, "my-custom-class", true).then(result => { return true; }, error => { return false; });
```
