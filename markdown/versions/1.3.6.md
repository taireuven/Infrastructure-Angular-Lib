## **מסמך הטמעה moh-package 1.3.6**

## תקציר:

גירסה זו מכילה תוספות קטנות ותיקוני באגים מגרסה 1.3.4 

**באגים שטופלו:**

1. Radio Buttons
    * איפוס/ ניקוי הבחירה כאשר מכניסים ערך null.  
    * תיקון הנראות כאשר יש טקסט ארוך.
    *  תיקון עיצוב הרכיב עבור explorer.
2. File Upload - הצגת הודעות השגיאה באקספלורר.
3. Datepicker
    * הצגת הודעות השגיאה בהזנת תאריך גדול/ קטן מידי.
    * הוספת בדיקות הולידציה הבסיסיות של הרכיב גם כאשר לא נשלחו ולידציות בקלט.

**היכולות שנוספו:**

1. התווסף interceptor (עבור קריאות httpClient) המוסיף header בשם App-Language המכיל את קוד השפה הנוכחית, לכל request  שנשלח.  
Interceptor זה מגיע עם כל ה  services האחרים ע"י הקוד שמוגדר ב  app module:  
MohPackageModule.forRoot()
2. wizard - למחלקה WizardStep נוספה ירושה מהמחלקה LabelBase.

## הטמעה:

1.	להסיר את ההוספה של ה header ב interceptor שנמצא בפרויקט.
2. wizard - כל רכיב היורש מ wizardStep צריך כעת להוסיף שליחה של Injector ל super כך:  
    ```typescript
    constructor(injector: Injector) {
        super(injector);
    }
    ```
3. RadioButtonGroup - שינוי "displyFeild" input  ל- "displayField".


## שימוש ביכולות:

**Wizard Route with Translation**

כל רכיב המשמש כ step יורש מהמהחלקה WizardStep, לחלקה זו נוספה ירושה מהמחלקה LabelBase.  
כך שבכל רכיב step יהיה ניתן להשתמש במאפיינים ובפונקציות של LabelBase.  
לדוגמה:  
```html
<div>{{getLabelText('key')}}<div>
```
